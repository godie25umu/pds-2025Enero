### **Descripción**

Una aplicación destinada a la adquisición de conocimientos de cualquier ámbito, de forma relajada y amena.

---

Casos de Uso: https://github.com/godie25umu/pds-2025Enero/blob/main/casosdeUso.md

---

Modelado de dominio: https://github.com/godie25umu/pds-2025Enero/blob/main/ModeladoDeDominio.md


---

Esta aplicación tiene como objetivo facilitar el aprendizaje de los usuarios mediante un sistema interactivo basado en lecciones cortas, que combinan diferentes tipos de preguntas. Además, permite realizar un seguimiento del progreso del usuario a través de un sistema de logros, junto con un sistema de niveles y experiencia, haciendo que el uso de la aplicación resulte más motivador e interesante.

---

Cómo ejecutar la aplicación:

Deberás importar el proyecto a tu IDE. Recomendamos el uso de Eclipse 2021-03. Una vez hecho esto, tendras que ir a la clase "Main" dentro del paquete "ejecutable" y ejecutar la clase como un programa cualquiera.

Una vez iniciada la aplicación, se mostrará una ventana de Login. Tendrá que registrarse como nuevo usuario con su nombre de usuario y su contraseña (al menos 6 caracteres). Después de esto, podrá hacer login y acceder a todas las funcionalidades de la aplicación.
